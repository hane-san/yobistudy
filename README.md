# 六法ねこカード

六法・条文・短答メモをカードで復習するための静的HTML/JSアプリです。GitHub Pagesでそのまま公開できます。

## 今回の版

- Edgeの拡大ミニプレイヤー下でもカード本文が隠れにくいよう、上部プレイヤー安全余白を入れています。
- 下フッターが見にくくならないよう、フッターと長押しボタン周辺を少しだけコンパクトにしています。
- 六法×猫アイコン候補を5個入れています。`icon-candidates.html` で確認できます。

## カードを追加・差し替えする方法

カード本文は `cards/company.json` に入っています。

今後カードを追加するときは、ChatGPTにこう依頼してください。

> このメモを追加した差し替え用 `cards/company.json` を作って！

作成された `company.json` を、GitHub上の `cards/company.json` と差し替えればOKです。

## アイコンを変える方法

`assets/` の候補から使いたいSVGを選び、`assets/icon.svg` としてコピー・差し替えしてください。

## 操作

- 下ボタン：カテゴリ／条文番号／ランダム
- 上スワイプ：次のカード
- 下スワイプ：前のカード
- 右スワイプ：現在の範囲内でランダム
- 左スワイプ：全カードからランダム
- 長押し：本文を小さくし、タグを消して全文表示
